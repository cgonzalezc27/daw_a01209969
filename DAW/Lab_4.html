Preguntas conceptuales

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias

  • JavaScript no sirve para crear aplicaciones que se sostienen por sí mismas, mientras que Java sí (corren en máquinas virtuales).
  • JavaScript es comúnmente utilizado dentro de un HTML y le da interactividad al documento, diferente a Java cuya función no es para esto.
  • Los Plug-ins requeridos por cada uno de los lenguajes para su ejecución son diferentes.
  • La forma de compilación y de ejecución de los códigos es diferente, lo que hace que el debugging sea diferente en cada caso.

Similitudes

  • Tanto Java como JavaScript son lenguajes de programación.
  • Ambos lenguajes pueden ser ejecutados en exploradores. Sin embargo, las aplicaciones de Java han ido perdiendo popularidad por este medio.
  • Los dos lenguajes usan bibliotecas y frameworks que permiten reusar código.

¿Cómo se declara una variable con alcance local dentro de una función?

La variable se declara como normalmente se declararía una variable, solo que esta se crea dentro de la función, lo que hace que solo esté disponible dentro de ésta misma.
